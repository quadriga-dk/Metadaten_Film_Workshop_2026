# Metadaten Film Workshop 2026

This repository contains the schedule and information for the Metadaten Film Workshop 2026.

## GitHub Pages

This site is published via GitHub Pages. Visit the [workshop schedule](./index.md) to see the full program.

## Setup

To enable GitHub Pages for this repository:
1. Go to the repository Settings
2. Navigate to "Pages" section
3. Under "Source", select the branch you want to publish (e.g., `main` or `copilot/add-table-to-md-file`)
4. Click "Save"
5. The site will be available at: `https://quadriga-dk.github.io/Metadaten_Film_Workshop_2026/`
